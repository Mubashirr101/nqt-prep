## 📋 TCS NQT Structure

| Section | Questions | Time |
|---|---|---|
| Numerical Ability | 26 | 40 min |
| Verbal Ability | 24 | 30 min |
| Reasoning Ability | 30 | 50 min |
| Programming Logic | 10 | 15 min |
| Coding | 2 | 30 min |
| **Prime** (Advanced Quant + Reasoning) | extra | extra |

**Ninja** = clear the first 5 sections with a decent cutoff
**Prime** = higher cutoff + clear the advanced sections too

## 🛣️ Your Roadmap (from zero)

### Phase 1 — Python Basics Solidified (Week 1-2)
Before DSA, get comfortable with:
- Lists, dicts, sets, strings
- Loops, functions, recursion basics
- List comprehensions
- `sorted()`, `map()`, `lambda`

### Phase 2 — DSA for NQT (Week 3-5)
You don't need advanced DSA. Focus on:
- **Arrays & Strings** — most coding Qs are here
- **Basic Sorting** — bubble, selection (understand, not memorize)
- **Searching** — linear, binary search
- **Patterns & loops** — very common
- **Basic Math** — GCD, primes, factorials
- **Maybe:** Stack, Queue (basic concept)

### Phase 3 — Aptitude (Week 1-5, parallel)
Do this alongside coding:
- Quant: Number system, percentages, ratios, profit/loss, time-speed-distance, permutations & combinations
- Reasoning: Seating arrangement, blood relations, syllogisms, coding-decoding
- Verbal: Reading comprehension, grammar, sentence completion

### Phase 4 — Mock Tests (Week 6+)
- Full-length TCS NQT mocks
- Time yourself strictly
- Review every wrong answer

## ⚡ Reality Check

- TCS NQT coding Qs are **easy to medium** — not LeetCode Hard
- Python is **fully accepted** — no need to switch
- Prime is achievable **even for beginners** with 6-8 weeks of consistent prep
- The **aptitude sections matter a lot** — don't ignore them for coding
