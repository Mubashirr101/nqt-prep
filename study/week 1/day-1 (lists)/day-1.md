### 🐍 Python — Lists (1.5 to 2 hrs)

Work through these in order. Open your editor and actually type everything out, don't just read.

**Part 1 — The basics you should already know**

```python
nums = [3, 1, 4, 1, 5, 9, 2, 6]

print(nums[0])       # first element
print(nums[-1])      # last element
print(nums[1:4])     # slicing → [1, 4, 1]
print(len(nums))     # length
print(9 in nums)     # True
```

**Part 2 — Methods to know cold**

```python
nums.append(7)       # add to end
nums.remove(1)       # removes first occurrence of 1
nums.pop()           # removes and returns last element
nums.pop(0)          # removes element at index 0
nums.sort()          # sorts in place
nums.reverse()       # reverses in place
print(nums.count(1)) # how many times 1 appears
print(nums.index(5)) # index of 5
```

**Part 3 — Your actual practice tasks (write these yourself)**

1. Take a list of numbers. Find the max and min **without** using `max()` or `min()`.
2. Reverse a list **without** using `.reverse()` or slicing `[::-1]`. Use a loop.
3. Remove all duplicates from a list. Output should keep original order.
4. Count how many even numbers are in a list.

These are easy but they make you think. Don't skip them.
