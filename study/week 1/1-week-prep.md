Week 1 — June 8 to June 14
Focus: Python solidified + Quant basics running in parallel every day

Each day has a coding part and a 30 min aptitude part. Don't skip either.

---

Monday June 8 — Lists

Go through list operations properly. indexing, slicing, append, remove, pop, len, in, count, sort, reverse. Write small programs yourself, don't just read.

Practice: take a list of numbers, find the max without using max(), reverse it without reverse(), remove duplicates.

Aptitude (30 min): Number system — divisibility rules, LCM, HCF. Use IndiaBIX, just do 20 questions.

---

Tuesday June 9 — Strings

Strings are huge in TCS NQT. upper, lower, strip, split, join, replace, find, count, slicing, checking palindrome, checking anagram.

Practice: check if a string is palindrome, count vowels in a string, reverse words in a sentence.

Aptitude (30 min): Percentages — 20 questions on IndiaBIX.

---

Wednesday June 10 — Dicts and Sets

dict: keys, values, items, get, update, looping through a dict. Sets: add, remove, union, intersection, difference. Know when to use a set vs a list.

Practice: count frequency of characters in a string using a dict, find common elements between two lists using sets.

Aptitude (30 min): Ratios and Proportions — 20 questions.

---

Thursday June 11 — Functions + Basic Recursion

Write clean functions with parameters and return values. Then understand recursion — just the concept and basic examples. Factorial and fibonacci recursively, understand the call stack mentally.

Practice: write factorial both iteratively and recursively, fibonacci same way, write a function that checks if a number is prime.

Aptitude (30 min): Profit and Loss — 20 questions.

---

Friday June 12 — Loops deep dive + List Comprehensions

Nested loops, loop with enumerate, loop with range tricks. Then list comprehensions — they show up in interviews and make your code cleaner.

Practice: print all prime numbers up to 100, generate a list of squares of even numbers using list comprehension, flatten a 2D list.

Aptitude (30 min): Time Speed Distance — 20 questions.

---

Saturday June 13 — sorted, map, lambda + mixed practice

sorted() with key argument, map(), filter(), lambda functions. These are small but useful.

Then do 5 HackerRank easy problems today — just pick any from the Python or Problem Solving section. Don't overthink, just attempt.

Aptitude (30 min): Mixed revision — 30 questions on whatever felt weak this week.

---

Sunday June 14 — Revision + Light Practice

Go back over anything that felt unclear this week. Redo the practice problems from scratch without looking at your notes. See what you remember and what you don't.

Do 3 more HackerRank easy problems. Check your solutions after, read how others solved it too.

Aptitude (30 min): 20 mixed quant questions to end the week.

---

That's Week 1. It's not overwhelming but it's consistent. The goal is by Sunday you can comfortably write Python programs with lists, strings, dicts, functions, and basic recursion without googling every line.
